# HTML Portfolio

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A simple web development training portfolio project designed to practice semantic HTML tagging, layout configurations, and link references.

---

## 🛠️ Tech Stack

- **Structure:** HTML5
- **Styling:** CSS3

---

## 📁 Repository Directory Structure

```
html-portfolio/
├── assets/         # Static visual assets and styling sheets
├── public/         # Shared public documents and templates
└── index.html      # Main portfolio entry page markup
```
